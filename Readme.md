## 如何安装
- 使用前需要安装git
- 在开源网站复制以.git结尾的克隆地址
- 可以直接使用git拉取代码复制到自己的项目中
- 也可以是使用 Unity的PackageManager 中 Add from git URL 来以插件包的方式安装
